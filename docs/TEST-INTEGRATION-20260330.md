## 🐛 发现问题

### 问题 1: 艺术家列表未显示

**现象:** 音乐库页面艺术家标签页显示"暂无艺术家"

**可能原因:**
1. Navidrome API 返回格式与预期不符
2. 艺术家数据本身为空
3. 解析逻辑有误

**排查步骤:**
```javascript
// 建议在浏览器控制台执行
const store = useNavidromeStore()
await store.loadArtists()
console.log(store.artists)
```

**优先级:** 中

---

### 问题 2: 路由未实现

**现象:** 点击专辑/艺术家无法跳转详情页

**原因:** 路由 `/album/:id` 和 `/artist/:id` 未配置

**解决方案:**
1. 创建 AlbumDetail.vue 和 ArtistDetail.vue 组件
2. 在 router/index.ts 中添加路由配置

**优先级:** 低
