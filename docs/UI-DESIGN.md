## 1. 设计愿景

### 设计理念

> **"音乐是主角，界面是舞台"**

我们的设计哲学：
- 🎯 **内容优先** - 让音乐和专辑封面成为视觉焦点
- 🌙 **沉浸体验** - 深色主题为主，减少视觉干扰
- ✨ **微妙的精致** - 细节处见品质，不过度设计
- 🎵 **流动感** - 平滑的动画和过渡，如音乐般流畅

### 设计风格选择

**主风格:** **Modern Dark Minimal** (现代深色极简)

| 特征 | 描述 |
|------|------|
| **色调** | 深色背景 + 温暖强调色 |
| **排版** | 清晰的层次结构，大标题 |
| **空间** |  generous 留白，呼吸感 |
| **动效** | 微妙、流畅、有意义 |
| **质感** | 毛玻璃效果 + 细腻渐变 |

**设计灵感:**
- Spotify 的深色主题
- Apple Music 的卡片设计
- Tidal 的高端质感
- 网易云音乐的情感化设计

## 7. 动效设计

### 7.1 过渡动画

```css
/* 页面切换 */
.page-transition {
  animation: fadeIn 0.3s ease-out;
}

@keyframes fadeIn {
  from {
    opacity: 0;
    transform: translateY(8px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* 卡片悬停 */
.card-hover {
  transition: all 0.2s cubic-bezier(0.4, 0, 0.2, 1);
}

.card-hover:hover {
  transform: translateY(-4px);
  box-shadow: 0 12px 24px rgba(0, 0, 0, 0.4);
}

/* 按钮点击 */
.button-press {
  transition: transform 0.1s ease;

}
```

## 11. 设计原则总结

1. **音乐优先** - 界面服务于内容，不喧宾夺主

3. **温暖点缀** - 琥珀色作为强调，增加温度感
4. **流畅动效** - 如音乐般流畅的过渡和反馈
5. **细节精致** - 微妙的阴影、渐变、圆角体现品质
6. **响应式优先** - 从移动端开始设计，逐步增强
