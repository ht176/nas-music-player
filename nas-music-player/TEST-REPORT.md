## ⚠️ 注意事项

### Vue Router 警告

测试中出现以下警告是正常的：
```
[Vue Router warn]: No match found for location with path "/playlist/1"
[Vue Router warn]: No match found for location with path "/playlist/2"
```

这是因为 Sidebar 组件中的歌单路由 (`/playlist/1`, `/playlist/2`) 在测试路由配置中没有定义，但不影响测试结果。
