# 🎵 NAS 音乐播放器 - 产品需求文档 (PRD)

**版本:** 1.0  
**日期:** 2026-03-28  
**状态:** 草稿

---

## 1. 产品愿景

打造一个**轻量、快速、美观**的自托管音乐播放器，让用户能够随时随地访问自己的音乐库。

### 核心价值
- 🏠 **私有部署** - 音乐数据完全掌握在自己手中
- 📱 **多端访问** - PC、手机、平板无缝切换
- 🎨 **精美界面** - 现代化的 UI 设计，优秀的视觉体验
- ⚡ **快速响应** - 低延迟播放，流畅的交互体验

---

## 2. 目标用户

| 用户类型 | 特征 | 核心需求 |
|---------|------|---------|
| **音乐爱好者** | 拥有大量本地音乐收藏 | 方便管理和播放个人音乐库 |
| **NAS 用户** | 已有 NAS 设备，注重数据隐私 | 私有化部署，不依赖第三方服务 |
| **多设备用户** | 在手机、电脑、平板间切换 | 跨设备同步播放进度和歌单 |
| **极简主义者** | 不喜欢复杂功能 | 简洁界面，核心播放功能 |

---

## 3. 用户故事 (User Stories)

### 🎯 P0 - 核心功能 (Must Have)

| ID | 用户故事 | 验收标准 |
|----|---------|---------|
| US-01 | 作为用户，我想浏览我的音乐库 | 能按歌手、专辑、歌曲、歌单查看 |
| US-02 | 作为用户，我想播放歌曲 | 点击歌曲即可播放，支持暂停/继续 |
| US-03 | 作为用户，我想控制播放 | 支持上一首/下一首/进度条拖动 |
| US-04 | 作为用户，我想调节音量 | 支持音量滑块调节 |
| US-05 | 作为用户，我想创建歌单 | 能创建、编辑、删除自定义歌单 |
| US-06 | 作为用户，我想搜索音乐 | 支持按歌名、歌手、专辑搜索 |
| US-07 | 作为用户，我想在手机上看 | 移动端界面适配良好 |

### 📌 P1 - 重要功能 (Should Have)

| ID | 用户故事 | 验收标准 |
|----|---------|---------|
| US-08 | 作为用户，我想看到歌词 | 播放时显示同步歌词 |
| US-09 | 作为用户，我想收藏喜欢的歌曲 | 支持红心收藏功能 |
| US-10 | 作为用户，我想查看播放历史 | 能查看最近播放的歌曲 |
| US-11 | 作为用户，我想随机播放 | 支持随机播放模式 |
| US-12 | 作为用户，我想循环播放 | 支持单曲循环、列表循环 |
| US-13 | 作为用户，我想在后台播放 | 切到后台/锁屏后继续播放 |

### 💡 P2 - 锦上添花 (Nice to Have)

| ID | 用户故事 | 验收标准 |
|----|---------|---------|
| US-14 | 作为用户，我想看到专辑封面 | 显示专辑艺术图 |
| US-15 | 作为用户，我想导入外部歌单 | 支持导入 Spotify/网易云歌单 |
| US-16 | 作为用户，我想分享歌曲 | 生成歌曲分享链接 |
| US-17 | 作为用户，我想设置播放质量 | 支持选择音质（标准/高/无损） |
| US-18 | 作为用户，我想看到播放统计 | 查看播放次数、时长统计 |

## 5. 技术约束

| 约束项 | 说明 |
|-------|------|
| **前端框架** | Vue 3 + TypeScript + Vite |
| **多端适配** | 响应式设计，支持 PC 和移动端 |
| **后端接口** | 需要 NAS 提供音乐文件访问 API |
| **音频格式** | 支持 MP3、FLAC、AAC、WAV 等常见格式 |
| **浏览器兼容** | Chrome、Safari、Firefox 最新版 |
| **性能要求** | 首屏加载 < 2s，播放延迟 < 100ms |

---

## 6. 非功能需求

### 🚀 性能
- 首屏加载时间 < 2 秒
- 歌曲切换延迟 < 100ms
- 搜索响应时间 < 500ms

### 📱 响应式
- 支持断点：320px、768px、1024px、1440px
- 触摸目标 ≥ 44×44px
- 支持横屏和竖屏

### 🔒 安全
- 支持用户认证（可选）
- API 访问需要鉴权
- 不暴露 NAS 其他文件

### ♿ 可访问性
- 支持键盘导航
- 足够的颜色对比度
- 屏幕阅读器友好

## 9. 风险与缓解

| 风险 | 影响 | 概率 | 缓解措施 |
|------|------|------|---------|
| NAS API 不稳定 | 高 | 中 | 添加重试机制和缓存 |
| 大音乐库加载慢 | 中 | 中 | 分页加载和虚拟滚动 |
| 移动端兼容问题 | 中 | 低 | 早期多设备测试 |
| 音频格式不支持 | 低 | 低 | 使用浏览器原生支持格式 |
